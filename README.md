# PotionCrossbreed

Плагин для Paper 1.21.8, позволяющий скрещивать зелья на наковальне.

## Возможности

- **Скрещивание зелий одного уровня** на наковальне (1+1→2, 2+2→3, максимум 3)
- **Скрещивание разных зелий** — эффекты объединяются в одно зелье
- **Стоимость:** 5 уровней опыта за каждое скрещивание
- **Автоматическое название:** "Скрещенное зелье" для разных зелий, обычное название с уровнем для одинаковых
- Работает с обычными, взрывными и туманными зельями

## Установка

1. Скачайте JAR из [GitHub Actions](../../actions)
2. Поместите в папку `plugins/` вашего Paper сервера
3. Перезагрузите сервер

## Использование

1. Откройте наковальню
2. Положите зелье в первый слот
3. Положите зелье того же уровня во второй слот
4. Заберите результат (стоит 5 уровней опыта)

### Примеры

| Первое зелье | Второе зелье | Результат |
|---|---|---|
| Зелье силы I | Зелье силы I | Зелье силы II |
| Зелье силы II | Зелье силы II | Зелье силы III |
| Зелье силы I | Зелье скорости I | Скрещенное зелье II (сила + скорость) |

## Сборка

Проект собирается автоматически через GitHub Actions. Скачайте готовый JAR из вкладки Actions.
